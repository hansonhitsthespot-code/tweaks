XMP / DOCP Profile

Essential

Unlocks your RAM's advertised speed. Most RAM ships underclocked by default to ensure compatibility. Enabling XMP lets it run at the speed you paid for.

Performance Gain: 10-30% boost in memory-intensive tasks, games, and compilation times

Above 4G Decoding

High Impact

Removes the 4GB address space limitation for PCIe devices. Critical for modern high-end GPUs, especially in multi-GPU configurations or compute workloads.

Performance Gain: 5-15% in GPU-bound scenarios, eliminates memory bottlenecks for large VRAM operations

Fast Boot Configuration

Quality of Life

Reduces POST time by skipping unnecessary hardware checks and compatibility testing.

Configuration Steps

1
Navigate to Boot section in BIOS
2
Set Fast Boot to Enabled
3
Disable Full Screen Logo for faster visual feedback
4
Consider disabling POST delay if available
Performance Gain: 20-50% reduction in boot time (typically 5-15 seconds faster)

Power Management Tuning

Advanced

Configure CPU and PCIe power states for either maximum performance or power efficiency.

CPU C-States

  • Location: CPU Configuration
  • Performance: Disable all C-States
  • Efficiency: Enable C6/C7 states
  • Impact: 5-10W idle power difference

PCIe ASPM

  • Location: PCIe Configuration
  • Performance: Disabled
  • Efficiency: L1 or Auto
  • Impact: Reduced GPU latency when disabled
âš  Power Consumption Trade-off
Disabling power-saving features increases idle power draw by 10-20W. For desktops, this is negligible. For laptops, expect reduced battery life.
